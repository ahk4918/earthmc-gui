Features
public

Nations & Towns

Browse all nations and towns, filter by residents, balance, or blocks. Cross-navigate between mayors, kings, and capitals.

wifi

Live Online List

See who's online now. Resident lists show green dots next to anyone currently on the server.

track_changes

Player Tracker

Watch specific players and get notified in the activity log when they join or leave.

storefront

Shop Manager

View your shops, stock levels, and pricing. Requires an EarthMC API key.

search

Universal Search

Hit Enter in the search bar to find any player, nation, or town instantly.

system_update

Auto Updates

Checks for new versions on launch and updates in the background.

Install
1

Download for your platform

Click the button above to download the correct version for your OS.

2

Install

Linux: sudo dpkg -i EarthMC-GUI-*.deb or double-click.
Windows: Run the .exe installer.
macOS: Double-click the .pkg and follow the prompts.

3

Optional: add your API key

Click the settings icon in the top-right to add your EarthMC API key for shop access.
